🚀 Getting Started

Quick Setup Instructions

  1. Install required packages: pip install -r requirements.txt
  2. Install DirectML support: pip install torch-directml
  3. Download and install the latest AMD GPU drivers
  4. Run the optimized application: python app_gpu_optimized_new.py
  5. Access the GPU-optimized page at http://localhost:8000/gpu-optimized

Note: The application will automatically detect your AMD GPU and enable GPU acceleration.

💡 Tips for Best Performance

Optimization Recommendations:

  • Close other GPU-intensive applications while processing
  • Use the ultrafast preset for faster encoding (slightly larger files)
  • Process videos in batches during off-peak hours
  • Keep your GPU drivers updated for best performance
  • Monitor GPU temperature during long processing sessions

📞 Support & Troubleshooting

Common Issues & Solutions:

  • No GPU detected: Install torch-directml and update AMD drivers
  • Performance issues: Check GPU temperature and close background applications
  • Memory errors: Reduce video resolution or process in smaller batches
  • Installation problems: Ensure you have Windows 10/11 with DirectML support
